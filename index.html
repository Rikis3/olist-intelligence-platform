Revenue vs SLA Breach Trend
SELECT DATE_TRUNC(order_purchase_timestamp, MONTH) AS month, SUM(payment_value) AS revenue, ROUND(COUNTIF(delivery_status = 'late') / COUNT(*) * 100, 1) AS sla_breach_pct FROM mart_revenue JOIN fact_fulfillment USING (order_id) GROUP BY 1 ORDER BY 1;
GMV by Category
SELECT p.product_category_name_english AS category, ROUND(SUM(oi.price) / 1e6, 2) AS gmv_millions, ROUND(SUM(oi.price) / SUM(SUM(oi.price)) OVER() * 100, 1) AS pct FROM fact_orders fo JOIN dim_products p USING (product_key) JOIN order_items oi USING (order_id) GROUP BY 1 ORDER BY 2 DESC LIMIT 8;
Monthly GMV with 3-Month Moving Average
SELECT month, revenue, AVG(revenue) OVER (ORDER BY month ROWS BETWEEN 2 PRECEDING AND CURRENT ROW) AS ma_3 FROM mart_revenue_monthly ORDER BY month;

Delivery Funnel — Where Delays Originate
SELECT AVG(DATE_DIFF(shipping_limit_date, order_purchase_timestamp, DAY)) AS seller_dispatch, AVG(DATE_DIFF(carrier_pickup, shipping_limit_date, DAY)) AS carrier_pickup, AVG(DATE_DIFF(delivered_date, carrier_pickup, DAY)) AS in_transit FROM fact_fulfillment WHERE delivery_status IS NOT NULL;

Cohort Retention Heatmap
WITH first_order AS ( SELECT customer_id, DATE_TRUNC(MIN(order_purchase_timestamp), MONTH) AS cohort FROM fact_orders GROUP BY 1 ) SELECT cohort, DATE_DIFF(DATE_TRUNC(o.order_purchase_timestamp, MONTH), cohort, MONTH) AS month_n, COUNT(DISTINCT o.customer_id) AS customers FROM fact_orders o JOIN first_order f USING (customer_id) GROUP BY 1, 2;
RFM Customer Segments
WITH rfm AS ( SELECT customer_id, DATE_DIFF(CURRENT_DATE(), MAX(order_purchase_timestamp), DAY) AS recency, COUNT(DISTINCT order_id) AS frequency, SUM(payment_value) AS monetary, NTILE(4) OVER (ORDER BY DATE_DIFF(...) DESC) AS r_score, NTILE(4) OVER (ORDER BY COUNT(*)) AS f_score, NTILE(4) OVER (ORDER BY SUM(payment_value)) AS m_score FROM mart_customer_health GROUP BY 1 ) SELECT CASE WHEN r+f+m >= 10 THEN 'Champions' WHEN r+f+m >= 8 THEN 'Loyal' ... END AS segment, COUNT(*) AS customers FROM rfm GROUP BY 1;

Strategic Recommendations
Data-backed action items to recover revenue and improve CSAT
Based on root cause analysis, logistics-driven churn costs $2.4M/year. 60% of SLA breaches originate at seller dispatch, not carrier transit. The Northeast region accounts for 38% of 1-star reviews despite only 14% of order volume.
1
Implement Seller Dispatch Penalty System
Penalize sellers who fail 48-hour dispatch SLAs. Could reduce SLA breaches by ~35%.
Impact: +2.1pp CSAT · $840K recovered
2
Open Northeast Cross-Docking Fulfillment Hub
Hub in Bahia for North/NE corridor. Cuts lead times from 21→12 days, freight costs by 18%.
Impact: -6 day lead time · $1.1M LTV recovered
3
Win-Back Campaign for Late-Delivery Churned Customers
Target SLA-breached users from the last 6 months with 15% discount. Recovers ~12% of churned high-value users.
Impact: +3.2K customers · $460K annual revenue
